От разработчика

GymStats - это проект, совмещающий в себе функции сайта для спортзала и электронного дневника тренировок, разработанный при помощи Java и JavaScript.

Этот сайт состоит из двух частей: первая - "лицевая", выполненная при помощи HTML и CSS, выполняет исключительно косметическую функцию. Вторая часть - "задняя", состоящая из backend`а. Сделана при помощи Java Spring, JavaScript, MySQL, Thymeleaf и прилегающих инструментов.

На сайте реализовано деление по ролям: "основатель", "менеджер", "тренер" и "пользователь", у каждой роли есть свои возможности.

Основатель назначает менеджеров, которые могут регистрировать аккаунты для пользователей, купивших абонемент, а также создает аккаунты для тренеров, которым может назначать учеников.

Каждому зарегистрированному пользователю доступен их профиль, в котором реализована система дневника тренировок. Занимающиеся могут отмечать выполненные упражнения, записывая их в таблицу, а также, могут сделать себе программу тренировок, тогда дневник будет заполняться автоматически. Также, каждый желающий пользователь может посмотреть статистику по любому выполненому им упражнению.

Тренеры, помимо того, что будут отображаться на лицевой части сайта на вкладке "Персонал", также могут назначать программу тренировок своим ученикам. Для этого тренер должен перейти в свой профиль и выбрать соответствующего пользователя в разделе "Ученики".
